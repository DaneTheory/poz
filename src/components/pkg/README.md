# README

### name

- Type: `string`

Project's name

### description

- Type: `string`

Project's description

### username

- Type: `string`

User's github name.

### email

- Type: `string`

User's email.

### compile

- Type: `boolean`

Whether to use `babel`

### cli

- Type: `boolean`

Whether to init a `cli`, use `egoist/cac`

### ut
 
- Type: `boolean`
 
Whether to init an UT framework, use `jest`

### eslint 

- Type: `String`

```js
eslint = 'disable' // disable eslint
eslint = 'xo' // use 'xo'
eslint = 'standard' // use 'standard'
```

### poi 

- Type: `boolean`
 
Whether to use `poi`


### nodeVersion

- Type: `String`
 
node version, default is `4`
